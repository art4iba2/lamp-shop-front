# LampStore — пользовательская часть интернет-магазина

React-приложение интернет-магазина ламп. Проект реализует пользовательскую часть сайта: главную страницу, каталог товаров, карточку товара, корзину, оформление заказа, подтверждение заказа, контакты и отзывы покупателей.

## Используемые технологии

- React
- Vite
- React Router DOM
- Local Storage
- Vitest
- React Testing Library

## Установка проекта

Перейдите в папку проекта:

```bash
cd shop-client
```

Установите зависимости:
```npm install```

## Запуск проекта
```npm run dev```

## Запуск тестов

```npm run test```
